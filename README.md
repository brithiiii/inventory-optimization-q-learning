# 📦 Inventory Level Optimization using Q-Learning

## 🚀 Project Overview
This project uses Reinforcement Learning (Q-Learning) to optimize inventory management under uncertain demand.

The model learns the best ordering strategy to minimize:
- Holding costs  
- Stockout penalties  
- Ordering inefficiencies  

---

## 🧠 Key Concepts
- Q-Learning (Off-policy TD Control)
- Reinforcement Learning
- Markov Decision Process (MDP)

---

## ⚙️ Problem Setup
- **State:** Inventory level  
- **Action:** Order quantity  
- **Reward:** Profit - holding cost - shortage cost
